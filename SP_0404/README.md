## 2️⃣ LLM(Large Language Model)의 원리와 활용

- **기술 선도 기업 동향**

## 3️⃣ 챗봇의 종류와 비교 분석

- **자동화 흐름 소개**  
  반복되는 서류 작업을 AI로 간소화
- **전문성 있게 쓰는 문장 구성 노하우**  
  프롬프트로 AI에게 구체적 지시 내리는 팁
